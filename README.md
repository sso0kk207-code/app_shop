# SFMShop

Интернет-магазин для продажи товаров.

## Описание
Проект для изучения Python на практике.

SFMshop разработан исполняя все стандарты проеткирования. В ходе работы было выполнено ...

## Как установить
1. Клонировать репозиторий

2. Заполнить свой .env данными для подключения к базе данных от postgresql и redis по примеру .env.example

3. Развернуть сервер с помощью команды: docker-compose up -d

4. Перейти на веб страницу по ссылке вашего сервера (в случае развертывывания на личном компьютере, можно зайти по ссылке localchost)
